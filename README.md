# lucasmigueis.com

My personal site, deployed at [lucasmigueis.com](https://lucasmigueis.com).

The site code lives in `src/features/`. Each section of the page is its own folder. Translations are in `messages/`, theme tokens in `src/app/globals.css`.

## Stack

Next.js 16, TypeScript, Tailwind 4. next-intl for the EN + PT-PT split, motion for animations, next-themes for dark mode.

## Running it

```bash
pnpm install
pnpm dev
```

Set `NEXT_PUBLIC_APP_URL` in `.env.local` first (see `.env.example`).
